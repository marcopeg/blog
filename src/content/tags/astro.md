Astro is a modern static site generator that helps you build faster websites with less client-side JavaScript. It's perfect for content-focused sites like blogs, documentation, and marketing pages.

## Why Astro?

- **Zero JavaScript by Default**: Ship only what you need
- **Island Architecture**: Partial hydration for better performance
- **Framework Agnostic**: Use React, Vue, Svelte, or plain HTML
- **Built for Content**: Great for blogs and documentation sites
